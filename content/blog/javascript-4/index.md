# 1. 반복문

while과 for문이 있다. 

## 1.1. while

다른 언어에서와 똑같다. 다음과 같은 형식을 하고 condition이 참일 동안 본문을 실행한다.

```js
while(condition){
    // 본문
}
```

condition의 평가값은 불린으로 변경되고 그 값이 true일 시 while 본문을 실행한다.
