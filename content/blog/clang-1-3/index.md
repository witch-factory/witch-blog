# 1. for문의 도입

같은 동작을 하는 코드도 다양한 방식으로 짤 수 있다. 가령 아까와 똑같은 동작을 하는 코드를 for문을 이용하여 짜 보자.

```c
#include <stdio.h>

/* 섭씨 - 화씨 온도 변환 표를 출력하는 코드 */

main() {
    int fahr;

    for (fahr = 0; fahr <= 300; fahr = fahr + 20) {
        printf("%3d %6.1f\n", fahr, (5.0 / 9.0) * (fahr - 32.0));
    }
}
```

이를 컴파일러를 통해 실행시켜 보면 1.2에서 작성했던, while문을 이용하는 코드와 같은 결과를 출력한다. 그러나 몇 가지의 변화가 있다. 

일단 화씨 온도를 나타내는 `fahr`변수를 제외하고는 모든 변수가 다 사라졌다. `lower` 와 `upper`변수를 for문에 집어넣고, 환산식도 계산해서 변수에 대입하는 것이 아니라 `printf`문의 인수로 직접 넣어 주었다.

여기서 알 수 있는 하나의 사실은, 특정 타입의 변수의 값을 사용할 수 있는 곳에서는 그 결과가 같은 타입이면 어떤 복잡한 계산식이라도 들어갈 수 있다는 것이다. 가령 `printf`의 세번째 인수에는 실수형 값이 들어갔는데, 이게 단일 실수형 변수이든 아니면 여러 변수들을 조합하고 계산해 나온 실수형 값이든 상관없다는 것이다.

변수를 줄인 것은, 이 정도로 메모리를 절약해야 하는 일은 거의 없기 때문에 성능상으로는 큰 의미가 없다. 하지만 코드가 많이 간단해졌다. 물론 변수명이 숫자의 용도를 나타내는 역할도 있다는 걸 생각하면 알아보기 힘들어졌다고 할 수도 있겠다. 하지만 이 또한 뒤에서 나올 `#define` 을 사용하면 해결할 수 있는 문제이다.
